Features

Everything you need for on-device AI

Fully Private

Every inference call runs on-device. Prompts, images, and documents never touch a server or leave the device.

GPU-Accelerated

MLX backend runs entirely on Apple Silicon GPU. Up to 45 tok/s for 1.7B models on iPhone 15 Pro.

Layer Streaming

Run 7B models on a 6 GB iPhone with peak RAM under 750 MB. The OS pages weights from disk on demand.

Swift 6 API

@MainActor-isolated, fully async/await with AsyncThrowingStream for token streaming. Zero data races by design.

Voice Pipeline

Full-duplex voice with on-device STT, LLM streaming, and sentence-level TTS pipelining. Zero network calls.

Document RAG

Index PDF, DOCX, and images locally. FTS5 keyword pre-filter plus cosine re-ranking. No external database.

Text generation

MLX backend. GPU-accelerated on Apple Silicon, 20–45 tok/s.

ContentView.swift 
import AuraCore

// One-liner
let reply = try await AuraLocal.chat("Summarize my expenses")

// Reusable instance — model loaded once
let llm = try await AuraLocal.text(.qwen3_1_7b) { progress in
    print(progress)  // "Fetching model: 42%"
}

// Streaming
for try await token in llm.stream("Explain this transaction") {
    print(token, terminator: "")
}

// With system prompt
let reply = try await llm.chat(
    "What is the VAT rate?",
    systemPrompt: "You are a finance assistant."
)

Large model — GGUF

llama.cpp backend. Download tracked automatically by ModelManager.

ContentView.swift 
import AuraCore

// ModelManager handles download + backend selection
let llm = try await ModelManager.shared.load(.llama3_1_8b_gguf)

// Observe state in SwiftUI
@ObservedObject var manager = ModelManager.shared

switch manager.state(for: .llama3_1_8b_gguf) {
case .idle:                       // not started
case .downloading(let progress):  // "Downloading: 62%"
case .loading:                    // initializing session
case .ready:                      // ready for inference
case .failed(let error):          // handle error
}

// Same streaming API regardless of backend
for try await token in llm.stream("Write a short story.") {
    print(token, terminator: "")
}

Voice pipeline

On-device STT → LLM → TTS. Sentence-level pipelining, zero network calls.

VoiceView.swift 
import AuraVoice

// Drop-in button — manages its own session
VoiceButton(llm: llm)

// Full control with an external session
@StateObject var session = VoiceSession(llm: llm)

VStack {
    Text(session.transcript)   // live STT transcript
    Text(session.response)     // live LLM response
    VoiceButton(session: session)
}

// Manual control
try await session.startListening()
session.interrupt()   // stop TTS mid-sentence
session.cancel()      // cancel everything

Document RAG

FTS5 + cosine re-ranking in SQLite. No external vector database required.

LibrarySetup.swift 
import AuraDocs

let library = DocumentLibrary.shared
await library.configure(embeddingProvider: AutoEmbeddingProvider(), llm: llm)
try await library.open()

// Index a PDF
try await library.add(url: pdfURL) { progress in
    print(progress)  // "Embedding Contract: 67%"
}
await library.refreshCorpus()

// Ask a question
let answer = try await library.ask("What is the total contract value?")
print(answer.text)

for source in answer.sources {
    print("[\(source.documentTitle) p.\(source.pageNumber)]")
}
